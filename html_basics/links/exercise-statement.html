Exercise

Let's practice what we've learned.

  • Create a project containing
  • 2 files in main directory (index.html and contact.html)
  • 1 directory named: files
  • inside that directory add 2 images of your choice and 1 file named images.html, which will list the images

Inside each .html file, you must place:

  1. Navigation menu with an unordered <ul></ul> list <li></li>
  2. title <h1></h1> of page
  3. One more paragraphs <p></p> with page information

Navigation

For the navigation menu use the tag <nav></nav> and place the unordered list as tag content.

The content of the link must be the name of the .html page that exists in the project, and for each page, we will have a link.

When we click on the link, we should be directed to the page clicked.

Page images.html

As content of this page, in addition to what was requested, also add

  1. The two images you have in the folder;
  2. Use the p tag to place the img tag.

Page contact.html

As the content of this page, in addition to what was requested, also add:

  1. Insert your contact information: email and phone;
  2. Each information must be inside a link that, when clicked, will open the respective contact (email or phone)
